# Практична робота "Масиви, вирази, керування виконанням програми"
## Завдання
Визначення "нещасливих" номерів бойової техніки.

### Опис завдання
Американці вважають нещасливим число **13**, китайці — **4**.  
Під час спільних навчань штаб української армії вирішив **не використовувати бойову техніку**, номер якої містить:

- цифру `4`, або
- число `13`.

#### Приклади номерів, що підлягають виключенню:

| Номер   | Причина |
|---------|---------|
| 40123   | містить `4` |
| 13313   | містить `13` |
| 12345   | містить `4` |
| 13040   | містить і `13`, і `4` |

В розпорядженні армії є **100000 одиниць техніки**.

## Мета роботи
Реалізувати програму, яка визначає, **скільки номерів доведеться виключити**, якщо перевірити всі номери від 1 до 100000.

## Деталі реалізації
У класі `Exercise` реалізовано статичний метод `Calculate`, який:

1. Приймає максимальний номер техніки.
2. Перебирає всі числа від 1 до 100000.
3. Перетворює кожен номер у рядок.
4. Перевіряє:
   - чи містить номер цифру `"4"`,
   - чи містить підрядок `"13"`.
5. Якщо так — номер додається до лічильника виключених.
6. Повертає загальну кількість таких номерів.
